🛡 개인정보처리방침

Last updated: 2025년 4월

MADNGINE - Flex Custom Work Time 확장 프로그램(이하 "확장")은 사용자의 개인정보를 보호하기 위해 아래와 같은 정책을 따릅니다.

1. 수집하는 개인정보 항목

본 확장은 어떠한 개인정보도 수집하거나 저장하지 않습니다. 모든 데이터는 사용자의 브라우저 내에서만 처리됩니다.

2. 데이터 이용 목적

확장은 https://flex.team 내 근무기록 페이지에서 다음의 정보를 사용자 화면에 표시하기 위해 사용됩니다:

  • 총 근무 시간
  • 실제 근무일 수
  • 연차 사용 현황

이 정보는 Flex 내부 API를 통해 사용자의 브라우저에서 직접 불러오며, 외부로 전송되지 않습니다.

3. 외부 전송 및 저장

사용자의 Flex 데이터를 외부 서버로 전송하거나 저장하지 않습니다. 데이터는 오직 사용자의 브라우저 메모리 내에서 실시간으로 처리됩니다.

4. 제3자 제공

어떠한 정보도 제3자에게 제공하거나 공유하지 않습니다.

5. 사용 권한 및 그 이유

확장은 다음과 같은 권한을 요청합니다:

  • 브라우저 탭의 flex.team 페이지 접근 및 DOM 분석
  • 로컬 저장소(localStorage) 접근 (설정 저장용)
